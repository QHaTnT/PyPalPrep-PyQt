## 1. 项目概述

### 1.1 项目简介

PyPalPrep 是一个基于 PyQt5 开发的 Python 学习教辅系统，专门针对计算机二级 Python 考试设计。系统集成了知识点学习、题库练习、模拟考试、代码编辑器、学习进度跟踪、错题本管理等功能，帮助用户系统地学习和备考。

### 1.2 项目目标

1. **系统化学习**：提供完整的 Python 知识点体系，覆盖计算机二级考试大纲
2. **练习巩固**：提供多种题型的练习题，帮助用户巩固知识点
3. **模拟考试**：模拟真实考试环境，帮助用户适应考试节奏
4. **进度跟踪**：记录用户的学习进度和成绩，便于分析和改进
5. **智能复习**：通过错题本和复习功能，帮助用户重点攻克薄弱环节

### 1.3 技术栈

- **编程语言**：Python 3.x
- **GUI框架**：PyQt5
- **数据库**：SQLite
- **AI助手**：DeepSeek API
- **数据可视化**：Matplotlib

## 2. 技术架构

### 2.1 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                      表示层 (UI Layer)                        │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐          │
│  │  登录界面   │ │   主窗口    │ │  各功能模块  │          │
│  └─────────────┘ └─────────────┘ └─────────────┘          │
├─────────────────────────────────────────────────────────────┤
│                      业务逻辑层 (Business Layer)             │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐          │
│  │  用户管理   │ │  学习管理   │ │  练习管理   │          │
│  └─────────────┘ └─────────────┘ └─────────────┘          │
├─────────────────────────────────────────────────────────────┤
│                      数据访问层 (Data Layer)                 │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐          │
│  │  数据库管理 │ │  数据加载器 │ │  代码执行器 │          │
│  └─────────────┘ └─────────────┘ └─────────────┘          │
├─────────────────────────────────────────────────────────────┤
│                      数据存储层 (Storage Layer)              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                    SQLite 数据库                     │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 设计模式

1. **单例模式**：数据库管理器（DatabaseManager）使用单例模式，确保整个程序只有一个实例
2. **MVC模式**：采用类似 MVC 的架构，将数据（Model）、界面（View）、逻辑（Controller）分离
3. **信号槽机制**：PyQt5 的核心通信机制，用于组件间通信
4. **延迟导入**：在函数内部导入模块，避免循环依赖

### 4.2 核心文件说明

#### main.py - 程序入口

**职责**：
- 创建 QApplication 实例
- 启动登录界面
- 登录成功后启动主窗口
- 进入事件循环

**关键函数**：
- `create_app()`: 创建并配置 QApplication
- `launch_main_window()`: 启动主窗口
- `run()`: 主函数，启动程序

#### config.py - 配置文件

**职责**：
- 存储全局配置信息
- 集中管理所有可配置项

**配置项分类**：
1. 路径配置：数据库路径、资源目录
2. 窗口配置：窗口大小、标题
3. 主题配置：颜色、字体
4. 数据配置：知识点分类、题目类型

#### database/db_manager.py - 数据库管理器

**职责**：
- 创建和管理数据库连接
- 执行 SQL 查询
- 封装所有数据库操作

**关键方法**：
- `connect()`: 连接数据库
- `disconnect()`: 断开连接
- `execute_query()`: 执行查询
- `execute_update()`: 执行更新
- `insert()`: 插入数据
- `verify_user()`: 验证用户

#### utils/data_loader.py - 数据加载器

**职责**：
- 从数据库加载各种数据
- 提供统一的数据访问接口

**关键方法**：
- `load_all_knowledge_points()`: 加载所有知识点
- `load_questions_by_category()`: 按分类加载题目
- `load_user_wrong_questions()`: 加载用户错题
- `get_user_statistics()`: 获取用户统计信息

#### utils/code_executor.py - 代码执行器

**职责**：
- 安全执行用户输入的 Python 代码
- 捕获代码输出
- 处理执行超时

**关键方法**:
- `execute()`: 执行代码
- `validate_code()`: 验证代码语法

## 7. 运行说明

### 7.1 环境要求

- **Python 版本**：Python 3.7+
- **操作系统**：Windows / macOS / Linux
- **依赖包**：PyQt5, matplotlib, requests

### 7.2 安装步骤

1. **克隆或下载项目**
   ```bash
   git clone https://github.com/your-username/PyPalPrep-PyQt.git
   cd PyPalPrep-PyQt
   ```

2. **安装依赖包**
   ```bash
   pip install -r requirements.txt
   ```

3. **初始化数据库**
   ```bash
   python scripts/init_data.py
   ```

4. **运行程序**
   ```bash
   python main.py
   ```

- **用户名**：1
- **密码**：1

## 8. 常见问题

### 8.1 安装问题

**Q: 安装 PyQt5 失败怎么办？**

A: 尝试使用以下命令：
```bash
pip install PyQt5 --user
# 或者使用国内镜像
pip install PyQt5 -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**Q: 导入 matplotlib 失败怎么办？**

A: 尝试重新安装：
```bash
pip uninstall matplotlib
pip install matplotlib
```

### 8.2 运行问题

**Q: 程序启动后界面显示异常怎么办？**

A: 尝试以下方法：
1. 检查屏幕分辨率设置
2. 更新显卡驱动
3. 尝试以管理员身份运行

**Q: 代码编辑器无法运行代码怎么办？**

A: 检查以下配置：
1. 确认 Python 环境正常
2. 检查代码执行超时设置
3. 查看控制台错误信息

### 8.3 数据问题

**Q: 如何重置所有学习数据？**

A: 在"个人主页"模块点击"重置学习记录"按钮，按照提示操作。

**Q: 如何备份数据库？**

A: 备份 `database/python_learning.db` 文件即可。

### 8.4 功能问题

**Q: AI 助手无法使用怎么办？**

A: 检查以下配置：
1. 确认网络连接正常
2. 检查 API 密钥是否有效
3. 查看控制台错误信息

**Q: 图表显示中文乱码怎么办？**

A: 确保系统安装了中文字体，程序已配置使用 Microsoft YaHei 字体。
